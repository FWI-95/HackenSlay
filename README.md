# HackenSlay

A Hack n Slay game built with [MonoGame](https://www.monogame.net/).

## Prerequisites

- MonoGame 3.8 or newer
- [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)

## Build

```bash
dotnet build
```

## Run

```bash
dotnet run
```
